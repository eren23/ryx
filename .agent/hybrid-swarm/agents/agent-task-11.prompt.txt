# Task: Standard library built-in functions and traits

Implement pkg/stdlib/core.go: type conversions (int_to_float, float_to_int, int_to_string, float_to_string, parse_int, parse_float returning Result), I/O (print, println, read_line, read_file, write_file), assertions (assert, assert_eq, panic with stack trace). Implement pkg/stdlib/string_ops.go: string_len, string_slice, string_contains, string_split, string_trim, string_chars, char_to_string, string_replace, string_starts_with, string_ends_with, string_to_upper, string_to_lower. Implement pkg/stdlib/array_ops.go: array_len, array_push, array_pop, array_map, array_filter, array_fold, array_sort, array_reverse, array_contains, array_zip, array_enumerate, array_flat_map. Implement pkg/stdlib/math_ops.go: abs, min, max (int/float), sqrt, pow, floor, ceil, round, random_int, random_float. Implement pkg/stdlib/io.go: file read/write on real filesystem. Implement pkg/vm/builtins.go: register all stdlib functions in VM, implement 6 built-in traits (Eq, Ord, Display, Default, Clone, Hash) with auto-implementation for primitives, arrays, tuples, structs, enums. Write stdlib_test.go covering all functions with boundary cases, empty inputs, Unicode, error conditions.

Target files: pkg/stdlib/core.go, pkg/stdlib/string_ops.go, pkg/stdlib/array_ops.go, pkg/stdlib/math_ops.go, pkg/stdlib/io.go, pkg/stdlib/stdlib_test.go, pkg/vm/builtins.go
