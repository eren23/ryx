# Task: Array stdlib expansion

Add 12 new array functions to pkg/stdlib/array_ops.go.

Functions using CallbackInvoker (higher-order):
- ArrayFind: 2 args (array, predicate), returns makeResultOk(first match) or makeResultErr("not found")
- ArrayAny: 2 args, returns BoolVal(true) on first truthy predicate result, false for empty
- ArrayAll: 2 args, returns BoolVal(false) on first non-truthy, true for empty

Numeric aggregation (1-arg, check all elements are Int or Float):
- ArraySum: returns Int if all Int, Float if any Float, IntVal(0) for empty
- ArrayMin: returns minimum via compareValues, errors on empty
- ArrayMax: returns maximum via compareValues, errors on empty

Slicing/chunking:
- ArrayTake: 2 args (array, n:Int), returns first min(n, len) elements, empty for n<=0
- ArrayDrop: 2 args, removes first n elements
- ArraySlice: 3 args (array, start:Int, end:Int), supports negative indices, clamps to bounds
- ArrayChunk: 2 args (array, size:Int), errors if size<=0, returns array of arrays

Other:
- ArrayUnique: 1 arg, O(n²) dedup using Value.Equal, preserves first occurrence order
- ArrayJoin: 2 args (array of strings, separator string), uses resolveString on each element, returns joined string

See goal spec section 6 for exact implementations.

Target files: pkg/stdlib/array_ops.go
