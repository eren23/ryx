# Task: VM core with dispatch loop, fibers, scheduler, and channels

Implement pkg/vm/value.go: Value struct (Tag byte + Data uint64), HeapObject with ObjectHeader (TypeID, GCMark, Size), payload types (StringObj, ArrayObj, StructObj, EnumObj, ClosureObj with UpvalueCell open/closed semantics, ChannelObj with buffer/senders/receivers). Implement pkg/vm/fiber.go: Fiber struct (ID, CallFrame stack max 1024, Value operand stack max 65536, SP, State enum Running/Suspended/Blocked/Dead, BlockedOn, Error). Implement pkg/vm/channel.go: CSP channel semantics (buffered/unbuffered, send blocks when full, recv blocks when empty, rendezvous for cap=0, close semantics, for-in draining). Implement pkg/vm/scheduler.go: cooperative M:1 green thread scheduler (round-robin FIFO ready queue, 4096-instruction timeslice, blocked fiber management, deadlock detection when all fibers blocked). Implement pkg/vm/error.go: runtime error types with source location and stack trace builder. Implement pkg/vm/vm.go: VM struct, dispatch loop (switch on opcode, decode operands, execute, advance IP), all 68 instruction implementations, stack overflow/call depth/div-by-zero/index-OOB checks with source locations. Write vm_test.go: per-opcode unit tests, function calls, recursion, tail calls, closures with upvalue closing, data structures, pattern matching dispatch, channel operations, fiber scheduling, deadlock detection, error reporting with stack traces.

Target files: pkg/vm/value.go, pkg/vm/fiber.go, pkg/vm/channel.go, pkg/vm/scheduler.go, pkg/vm/error.go, pkg/vm/vm.go, pkg/vm/heap.go, pkg/vm/vm_test.go
