# Task: Filesystem stdlib expansion

Add 8 filesystem functions to pkg/stdlib/io.go.

1. FileExists: 1 arg (path string), returns BoolVal(os.Stat(path) == nil)
2. DirList: 1 arg, returns Result<[String], String> using os.ReadDir, returns entry names only
3. DirCreate: 1 arg, returns Result<Unit, String> using os.MkdirAll(path, 0755)
4. PathJoin: 1 arg (array of strings), uses filepath.Join
5. PathDirname: 1 arg, uses filepath.Dir
6. PathBasename: 1 arg, uses filepath.Base
7. PathExtension: 1 arg, uses filepath.Ext
8. FileSize: 1 arg, returns Result<Int, String> using os.Stat(path).Size()

New imports needed: "path/filepath"

Use existing resolveString and resolveArray helpers (same package). Use existing makeResultOk/makeResultErr helpers.

See goal spec section 8 for exact implementations.
