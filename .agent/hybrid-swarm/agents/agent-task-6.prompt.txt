# Task: Time and random stdlib functions

Create new file pkg/stdlib/time_ops.go with 5 functions.

Package-level state:
- var rng *rand.Rand with sync.Mutex protection
- init() function that seeds rng with time.Now().UnixNano()

Functions:
1. TimeNowMs: 0 args, returns IntVal(time.Now().UnixMilli())
2. SleepMs: 1 Int arg, errors if ms < 0, calls time.Sleep(Duration(ms)*Millisecond)
3. RandomSeed: 1 Int arg, resets rng = rand.New(rand.NewSource(seed)) under lock
4. RandomShuffle: 1 array arg, copies elements, Fisher-Yates shuffle using rng under lock, returns NEW array
5. RandomChoice: 1 non-empty array arg, returns random element using rng.Intn under lock, errors on empty array

IMPORTANT: The existing RandomInt and RandomFloat in math_ops.go use math/rand/v2 global state. These should ideally use the shared rng, but to avoid file conflicts, note this as a TODO in comments. The new random functions must use the package-level rng with mutex.

Use the existing resolveArray helper from array_ops.go (same package, so it's accessible).

Imports needed: "fmt", "math/rand", "sync", "time", "github.com/ryx-lang/ryx/pkg/vm"

Target files: pkg/stdlib/time_ops.go
