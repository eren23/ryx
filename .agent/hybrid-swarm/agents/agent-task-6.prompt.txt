# Task: Add draw image commands to graphics_image.go

Extend `pkg/stdlib/graphics_image.go` with DrawImageCmd and draw builtins.

Add to the existing file:
1. `DrawImageCmd` struct: `Handle int`, `X int`, `Y int`, `ScaleX float64`, `ScaleY float64`. Must implement the `DrawCommand` interface.
2. `DrawImageCmd.Execute(screen *ebiten.Image)`: look up image from `imageStore` by Handle, create `ebiten.DrawImageOptions`, apply `GeoM.Scale(ScaleX, ScaleY)` if not (1.0, 1.0), apply `GeoM.Translate(float64(X), float64(Y))`, call `screen.DrawImage(img, opts)`. Handle invalid handles gracefully (skip draw).
3. `GfxDrawImage(args []vm.Value, heap *vm.Heap) vm.Value`: extracts (handle: Int, x: Int, y: Int), creates `DrawImageCmd{ScaleX: 1.0, ScaleY: 1.0}`, appends to draw queue via `AppendCommand`.
4. `GfxDrawImageScaled(args []vm.Value, heap *vm.Heap) vm.Value`: extracts (handle: Int, x: Int, y: Int, sx: Float, sy: Float), creates `DrawImageCmd` with given scale.
5. Add both to `RegisterImageBuiltins`.
