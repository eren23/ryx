# Task: Type system with Hindley-Milner inference and trait resolution

Implement pkg/types/types.go: all type representations (IntType, FloatType, BoolType, CharType, StringType, UnitType, ArrayType, SliceType, TupleType, FnType, StructType, EnumType, ChannelType, TypeVar, TypeScheme with constraints). Implement pkg/types/inference.go: constraint generation (walk AST, assign fresh type vars, emit equality + trait constraints), union-find unification with occurs check, generalization at let/fn, monomorphization at call sites with fresh type vars. Implement pkg/types/traits.go: trait impl registry keyed by (TraitName, ConcreteType), generic impl matching by unification, coherence checking (at most one impl per pair), orphan rule. Implement pkg/types/checker.go: type checking driver that walks AST applying all documented type rules (if/else branch unification, match arm unification, binary op type requirements, array indexing, closure param inference, spawn body must return Unit, channel type consistency, for-in-range typing). Write types_test.go with comprehensive inference, trait, coherence, exhaustiveness, and monomorphization tests. Create golden test data in tests/testdata/typecheck/.

Target files: pkg/types/types.go, pkg/types/inference.go, pkg/types/traits.go, pkg/types/checker.go, pkg/types/types_test.go, tests/testdata/typecheck/inference_basic.vex, tests/testdata/typecheck/inference_generics.vex, tests/testdata/typecheck/traits.vex, tests/testdata/typecheck/exhaustiveness.vex, tests/testdata/typecheck/errors.vex, tests/testdata/typecheck/complex_types.vex, tests/integration/typecheck_golden_test.go
