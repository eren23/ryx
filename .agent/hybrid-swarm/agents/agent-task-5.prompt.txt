# Task: Math stdlib expansion

Add 16 new math functions to pkg/stdlib/math_ops.go.

All follow the existing pattern using toFloat() helper for auto-promotion.

Trigonometric (all 1-arg, return Float):
- Sin, Cos, Tan, Asin, Acos, Atan: wrap math.Sin/Cos/Tan/Asin/Acos/Atan(toFloat(args[0]))
- Atan2: 2 args, math.Atan2(toFloat(args[0]), toFloat(args[1]))

Logarithmic/Exponential (all 1-arg, return Float):
- Log, Log2, Log10, Exp: wrap corresponding math.* functions

Constants (0-arg, return Float):
- Pi: returns FloatVal(math.Pi)
- E: returns FloatVal(math.E)

Integer math:
- Gcd: 2 Int args, Euclidean algorithm with absolute values
- Lcm: 2 Int args, lcm(a,b) = |a|/gcd(a,b)*|b|, returns 0 if either is 0
- Clamp: 3 args (x, lo, hi). If all Int, use Int comparison. Otherwise promote to Float.

See goal spec section 4 for exact implementations and edge case handling.

Target files: pkg/stdlib/math_ops.go
