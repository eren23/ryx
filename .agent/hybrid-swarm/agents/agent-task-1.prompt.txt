# Task: ObjMap core types and heap allocation

Add MapObj heap type to pkg/vm/value.go and AllocMap to pkg/vm/heap.go.

In value.go:
1. Add ObjMap byte constant = 0x27 after ObjChannel = 0x26
2. Add mapEntry struct with Key and Value fields (both vm.Value)
3. Add MapObj struct with Buckets (map[uint64][]mapEntry) and Count (int)
4. Add exported ComputeHash(v Value, heap *Heap) (uint64, error) function that uses fnv.New64a() + hashValue, rejecting ClosureObj and ChannelObj as unhashable
5. Add MapObj methods: Get, Set, Delete, Contains, Keys, Values, Entries (see goal spec sections 1.2-1.4 for exact signatures and implementations)

In heap.go:
6. Add AllocMap() uint32 method that allocates a HeapObject with TypeID=ObjMap and Data=&MapObj{Buckets: make(map[uint64][]mapEntry), Count: 0}

Note: Must import "hash/fnv" in value.go. The mapEntry type and MapObj must be exported so stdlib/map_ops.go can reference them.

Target files: pkg/vm/value.go, pkg/vm/heap.go
