# Task: Create PNG wall texture assets

Create the directory `examples/assets/` and generate two minimal 32x32 PNG texture files: (1) `examples/assets/wall_stone.png`: 32x32 pixels with a gray brick pattern using alternating horizontal bands of colors #787878 and #646464. (2) `examples/assets/wall_moss.png`: 32x32 pixels with a green-gray pattern using alternating bands of #5A6E50 and #4A5E40. These can be generated using Go's image package in a small helper script, or any other method. The game gracefully falls back to solid-color rectangles if these files don't load (gfx_load_image returns -1), so simple placeholder images are fine.

Target files: examples/assets/wall_stone.png, examples/assets/wall_moss.png
